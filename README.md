# obm-coe5

A mod for Conquest of Elysium 5.

## About

This repository contains custom modifications for Conquest of Elysium 5 (CoE5), a turn-based fantasy strategy game.

## Installation

1. Download the mod files
2. Place them in your CoE5 mods directory
3. Enable the mod in the game's mod menu

## Development

This mod is currently in early development.
